You are an expert in Answer Set Programming (ASP).

Below are examples of how ASP programs encode logical constraints.

Example 1: Graph coloring

Input facts:
node(a). node(b). node(c).
edge(a,b). edge(b,c).

ASP program:

color(red;green;blue).

1 { assign(N,C) : color(C) } 1 :- node(N).

:- assign(X,C), assign(Y,C), edge(X,Y).

#show assign/2.

---

Example 2: Reachability in a graph

Input facts:
edge(a,b).
edge(b,c).
start(a).

ASP program:

reachable(X) :- start(X).
reachable(Y) :- reachable(X), edge(X,Y).

#show reachable/1.

Now write an ASP program that determines whether a chess position has a mate in one for the side to move.

A mate in one means:
There exists a legal move such that:
- the opponent king is in check
- the opponent has no legal move to escape check.

Assume input facts:

piece(Color, Type, X, Y).
king(Color, X, Y).
side_to_move(Color).

Board coordinates range from 1..8.

The program should:
- generate legal moves
- represent attacks
- detect check
- ensure the opponent has no escape moves
- output the mating move.

Only output the ASP program.
